## Einzelne Spalte ausgeben

<EditorLink
  link="https://editor.sqlverine.org/?db=Grundschule.db"
  sqlQuery="SELECT spaltenname FROM tabellenname"
  runBtn="hide"
></EditorLink>

Mit diesem Befehl werden alle Zeilen der Spalte "spaltenname" aus der Tabelle "tabellenname" ausgegeben.

### Beispiel

#### Befehl:

<EditorLink
  link="https://editor.sqlverine.org/?db=Grundschule.db"
  sqlQuery="SELECT vorname FROM schueler"
  runBtn="show"
></EditorLink>

#### Ergebnis:

| vorname |
| ------- |
| Paul    |
| Margot  |
| Mahmud  |
| Sabrina |
| Ayse    |

## Gezielt mehrere Spalten ausgeben

<CustomEditorLink
  fontSize=""
  border=""
  sqlQuery="SELECT/s spaltenname_1/c ,/s spaltenname_2/c ,/s .../c ,/s spaltenname_n/c FROM/s tabellenname/t"
></CustomEditorLink>

Hiermit können mehrere Spalten einer Tabelle angezeigt werden. Es werden immer alle Zeilen der selektierten Spalten angezeigt.

### Beispiel

#### Befehl:

<EditorLink
  link="https://editor.sqlverine.org/?db=Grundschule.db"
  sqlQuery="SELECT id, vorname FROM schueler"
  runBtn="show"
></EditorLink>

#### Ergebnis:

| id  | vorname |
| --- | ------- |
| 1   | Paul    |
| 2   | Margot  |
| 3   | Mahmud  |
| 4   | Sabrina |
| 5   | Ayse    |

---

## Alle Spalten ausgeben

<EditorLink
  link="https://editor.sqlverine.org/?db=Grundschule.db"
  sqlQuery="SELECT * FROM tabellenname"
  runBtn="hide"
></EditorLink>

Mit diesem Befehl werden alle Zeilen aller Spalten der Tabelle angezeigt.

### Beispiel

#### Befehl:

<EditorLink
  link="https://editor.sqlverine.org/?db=Grundschule.db"

  runBtn="show"
></EditorLink>
