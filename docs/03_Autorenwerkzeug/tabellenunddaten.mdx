# Tabellen erstellen

## Tabellen erstellen mit SQLverine
<img
  src={require('../assets/author4.png').default} 
/>

Eine neue Tabelle kannst du mit Hilfe des integrierten SQLverine Editors erstellen. Klicke dazu auf "SQLverine Editor". Hier kannst du normalen SQLite Code eingeben um eine neue Tabelle zu erstellen:

## Tabellen erstellen mit direkter Texteingabe
<img
  src={require('../assets/author5.png').default} 
/>

Alternativ kannst du auch in den Textmodus wechseln, indem du rechts auf den **Schalter "Aa"** klickst. Nun kannst du direkt SQLite Code eingeben um neue Tabellen zu erstellen.
