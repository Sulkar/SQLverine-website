# Daten hinzufügen

Im Tab **Daten bearbeiten** können Einträge der Datenbank bearbeitet und neu hinzugefügt werden.
<img
  src={require('../assets/author11.png').default} 
/>

Mit Hilfe des DropDown Menüs kann zwischen den verschiedenen vorhandenen Tabellen der Datenbank gewechselt werden. 

## Daten verändern
Daten können nach Auswahl der Tabelle verändert werden, indem man einfach in die gewünschte Zelle der Tabelle klickt. Nach der Änderung von Daten muss nur noch gespeichert werden.
<img
  src={require('../assets/author13.png').default} 
/>

## neue Daten hinzufügen
Neue Daten können einfach über die Schaltfläche **neue Zeile** hinzugefügt werden. Hat eine Spalte das Attribut **PRIMARY KEY**, wird diese Spalte sofort mit **auto** befüllt. Das heißt, dass nach dem Speichern eine automatische Nummerierung vorgenommen wird.
<img
  src={require('../assets/author14.png').default} 
/>

## Daten importieren
Daten können sehr einfach im CSV Format importiert werden. Dazu klickt man auf die Schaltfläche **CSV Daten importieren** und fügt die rohen Daten dort ein.
<img
  src={require('../assets/author15.png').default} 
/>

Gibt es in der Tabelle eine Spalte mit dem Namen **id** und hat diese Spalte das Attribut **PRIMARY KEY** kann diese Spalte ignoriert werden, damit die eingefügten Daten automatisch eine fortlaufenden **id** bekommen. Will man das nicht, muss man rechts oben die Checkbox **Spalte id ignorieren** abwählen.
