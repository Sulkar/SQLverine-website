# Neue Datenbank erstellen

<img
  src={require('../assets/author2.png').default} 
/>

Eine neue Datenbank erstellst du, indem du auf die Schaltfläche "Neue Datenbank" klickst. Daraufhin wird eine neue SQLite Datebank erstellt und geladen.

Die neue Datenbank kannst du anschließend umbenennen.

Nachdem die Datenbank neu angelegt wurde, wird gleich im Info-Tab die Option angezeigt, der neuen Datenbank eine Übungs- sowie eine Infotabelle hinzuzufügen. 

<img
  src={require('../assets/author3.png').default} 
/>

Klickst du auf "Tabellen anlegen" werden die beiden Tabellen angelegt. Dies ist jedoch optional und wird nur benötigt, wenn du interaktive Übungen der Datenbank hinzufügen willst.
