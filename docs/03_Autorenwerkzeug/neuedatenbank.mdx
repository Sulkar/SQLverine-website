Eine neue Datenbank erstellt man, indem man auf die Schaltfläche **"Neue Datenbank"** klickt. Daraufhin wird eine neue SQLite Datebank erstellt und geladen.

Die neue Datenbank kann anschließend umbenannt werden.

Nachdem die Datenbank neu angelegt wurde, wird im Info-Tab die Option angezeigt, der neuen Datenbank eine Aufgaben- sowie eine Infotabelle hinzuzufügen. 

<img className='customImageBorder'
  src={require('../assets/author3.png').default} 
/>

Klickt man auf **"Tabellen anlegen"** werden beide Tabellen angelegt. Dies ist optional und wird nur benötigt, wenn man interaktive Übungen der Datenbank hinzufügen will.
