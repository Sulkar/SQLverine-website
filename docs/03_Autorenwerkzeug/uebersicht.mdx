# Übersicht
Mit Hilfe des Autorenwerkzeugs kannst du einfach online Datenbanken erstellen, Daten importieren und sogar interaktive Aufgaben zusammenstellen. Die erstellten Datenbanken können dann wiederum in den SQLverine Editor eingelesen und weitergegeben werden.

Das **Authoringtool** ist in zwei verschiedene Bereiche eingeteilt:
<img
  src={require('../assets/author1.png').default} 
/>

1. Datenbankmenü
2. Editorbereich

## Datenbankmenü
<img
  src={require('../assets/author6.png').default} 
/>
Im Datenbankmenü kann zwischen verschiedenen Datenbanken mit Hilfe des Auswahlmenüs gewechselt werden.

Die Datenbank kann umbenannt werden. Genauso kann eine Datenbank geöffnet/hochgeladen oder die aktuelle Datenbank auf dem Computer gespeichert/heruntergeladen werden.

Auch neue leere Datenbanken lassen sich erzeugen.

## Editorbereich
Im Editorbereich können neue Aufgaben erstellt und durch diese navigiert werden. Es ist möglich Daten direkt zu verändern, zu importieren und zu erstellen.

Um Abfragen auf die Datenbank direkt ausprobieren zu können, ist der SQLverine Editor integriert.

### Info - Tab
<img
  src={require('../assets/author7.png').default} 
/>
Im Info Tab können Informationen zur Datenbank und zu den erstellten Aufgaben eingetragen werden. Diese werden in der Datenbank gespeichert und können im SQLverine angezeigt werden.

### Aufgaben - Tab
<img
  src={require('../assets/author8.png').default} 
/>

<img
  src={require('../assets/author9.png').default} 
/>
Im Aufgaben Tab können Aufgaben für die Datenbank erstellt werden. Aufgaben leiten den Lernenden durch die verschiedenen Tabellen einer Datenbank und können automatisches Feedback geben.


### Vorschau - Tab
<img
  src={require('../assets/author10.png').default} 
/>
Die Vorschau zeigt die ausgewählte Aufgabe an, so wie sie im SQLverine zu sehen ist.

### Daten bearbeiten - Tab
<img
  src={require('../assets/author11.png').default} 
/>
Daten der Datenbank können direkt im Daten bearbeiten Tab durchsucht und neu angelegt werden. Hier hat man die Möglichkeit, Daten einzeln einzugeben oder eine ganze Datenbasis direkt zu importieren. Auch ein Export der Daten ist möglich.
