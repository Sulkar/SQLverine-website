# Service

Die SQLverine Lernplattform ist ein Open Source Projekt, welches jedem interressierten Lernenden kostenlos zur Verfügung steht.

Neben diesem kostenlosen Diensten, können besondere Erweiterungen hinzugebucht werden, um die Entwicklung und das Hosting der Seite zu unterstützen.

Folgende Dienste stehen aktuell zur Verfügung:

<Admonition type="tip" icon="" title="Eigene Datenbanken hochladen">
  <p>
    Eigene Datenbanken können einfach mit dem{" "}
    <a href="https://author.sqlverine.org">SQLverine Autorenwerkzeug</a>{" "}
    erstellt werden. Sie bekommen einen <strong>SQLverine Account</strong>, mit
    dem eigens erstellte Datenbanken hier auf SQLverine hochgeladen und
    Lernenden zur Verfügung gestellt werden können.
  </p>
